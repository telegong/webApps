### Mermaid diagrams for data generation evolution

아래 코드는 바로 복사해 Mermaid 지원 도구나 에디터에 붙여서 벡터(SVG)로 렌더링할 수 있어요.

---

#### Flowchart: 1세대 → 2세대 → 객체 기반 3세대

```mermaid
flowchart LR
  A[1세대: 문서 전산화 중심] --> B[2세대: 디지털 모델링·시스템 입력 중심]
  B --> C[3세대: 객체·온톨로지 기반 자동 생성]

  subgraph Gen1[1세대]
    A1["문서 작성(Word/PDF)"]
    A2["스프레드시트(Excel)"]
    A3[수작업 입력]
    A1 --> A2 --> A3
  end

  subgraph Gen2[2세대]
    B1[CAD/BIM 모델링]
    B2[ERP/PM 직접 입력]
    B3[CDE 협업]
    B1 --> B2 --> B3
  end

  subgraph Gen3[3세대]
    C1["객체 조합(온톨로지)"]
    C2[규칙/법규 기반 자동 생성]
    C3["과거 프로젝트 학습(AI)"]
    C4[ERP/GIS/IOT 연결형 입력]
    C1 --> C2 --> C3 --> C4
  end

  A --> A1
  B --> B1
  C --> C1
```

#### Mindmap: 객체 기반 3세대의 구성 요소

```mermaid
mindmap
  root((객체 기반 3세대))
    객체 조합
      표준 공간/요소 라이브러리
      온톨로지 매핑
    규칙 기반 생성
      법규/코드 체크
      설계 가이드 자동 반영
    AI 생성
      사례 학습
      요구조건 → 초기안 자동 제안
    연결형 입력
      ERP 비용/수량
      GIS/토지 데이터
      IoT/현장 센서
    결과 활용
      비용·일정 자동 산출
      리스크 시뮬레이션
      실시간 대시보드
```

#### Gantt: 도입 로드맵 예시

```mermaid
gantt
  title Enterprise CDE 전환 로드맵
  dateFormat  YYYY-MM-DD
  section 준비
    현행분석온톨로지설계        :a1, 2026-01-01, 30d
    데이터정제표준화             :a2, after a1, 45d
  section 구현
    객체라이브러리구축           :b1, 2026-03-01, 40d
    규칙법규엔진통합            :b2, after b1, 30d
    AI초기안생성파일럿         :b3, after b2, 30d
  section 연결
    ERPGISIOT연동              :c1, 2026-05-01, 40d
    대시보드의사결정워크플로우    :c2, after c1, 30d
  section 운영
    현장파일럿피드백루프        :d1, 2026-07-15, 30d
    확산교육최소화전략          :d2, after d1, 20d
```

### 텍스트 기반 오픈소스/에디터 추천

- **Mermaid JS:** 텍스트 기반으로 플로우차트, 마인드맵, Gantt 등 다양한 다이어그램을 생성할 수 있는 오픈소스 생태계로, 코드 중심 생성과 시각 편집을 모두 지원합니다.
- **Text2Diagram:** Mermaid 문법을 그대로 사용하면서 협업/임베드/벡터(SVG) 내보내기를 지원하는 텍스트 기반 다이어그램 편집기입니다.

> 참고로 Mermaid 지원 에디터들은 Mindmap, Flowchart, ER, Gantt 등 여러 타입을 제공하며, 공유와 SVG 링크/내보내기 같은 워크플로우도 지원합니다.
